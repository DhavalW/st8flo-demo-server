The problem statement




Humans think visually, not in text.


Manually decoding text based code into a visual mental map,
is a time consuming and taxing activity,
but still required by every developer, whether collaborating on a project, or working solo

which leads to gaps in understanding, and therefore bugs & longer turnaround times



The Solution



What if


you never needed to build mental maps from code,

in the first place ?





St8Flo.js

Text is for machines | Visual, is for humans



Your app's flowchart, itself becomes the app !




Understand & design your code's structure visually




Edit the flowchart directly to change the app's behaviour.




No more need to decode and undertand heaps of text

Understand code flow more easily & with less errors



Most slowdowns and errors happen when you attempt to create / edit / debug code, without realising that you've missed out on some important architectural details.

Other key Features



Use with ANY JS framework
in the browser, server side
or even standalone mobile apps


St8Flo is designed to be as generic as possible,
and can be used with technologies like angular & node.js alike
and even with hybrid mobile app platforms like Ionic





Build entirely new apps / websites
OR integrate into your existing ones


St8Flo programs can be packed into independent modules, which can be integrated into your existing javascript code through a simple "require()"



Reduce spaghetti code & code bloat
